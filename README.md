# Teachvatory Studio

Static GitHub Pages app for reviewing generated assessment JSON.

Live app:
https://www.rrmaximiliano.com/teachvatory-studio/

Repository:
https://github.com/RRMaximiliano/teachvatory-studio

The published repository intentionally contains only the shareable app surface. Local source materials such as docs, plans, transcripts, audio files, and agent configuration are ignored.

To open locally:

```sh
open index.html
```
